# Embeddad-System

## Nock
* NFC를 사용한 도어락 시스템

## Develop Stack
* Client
  * Andorid 5.0(Lollipop)
  * Raspberry Pi3
* Server
  * Python Flask
* Database
  * MariaDB

## Installation
```
pip install -r requirements.txt
```

## Execution
```
python main.py
```

## URL
URL  | Page Name | Completed
------- | -------|-----------
